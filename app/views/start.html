Flexible Standards

We use the well-known web technology standards HTML5, CSS, JavaScript and node.js. This makes it easy to work with tatool and even simpler to find a developer.

Built for researchers

We focused on the functionality that you need to get your research done. Go from programming, configuring experiments to collecting data and analysing it in no time.

Reach everyone, everywhere

Thanks to our web technologies you can include participants from all around the world by telling them to simply start up their favourite web browser.
