statusPanelUtils

The statusPanelUtils service offers methods that allow you to interact with the Tatool status panels. The most common methods are used update the status panel values.

  • Status Panel Methods

Status Panel Methods

When to use the Status Panel Methods?
Tatool takes care of updating the Status Panel by its own or by the help of customized handlers. Usually you'll not have to manually update a status panel value. See the methods below to learn how and when to use them.

setFeedback(score) - (no return value)

Used to manually update the feedback status panel to display a correct or a wrong feedback. The feedback status panel is automatically updated as soon as a trial object is saved with the help of the dbUtils service. If you're not saving a trial or want to update the feedback at a different time within your Executable, you can use this method instead.

Parameters
score: integer value indicating the score (> 0 will show a correct feedback and 0 will show a wrong feedback) 

// update the feedback within an Executable
statusPanelUtils.setFeedback(1);
