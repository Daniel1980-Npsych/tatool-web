Documentation

Welcome to the Tatool documentation, your resource to learn everything you need to know to start using Tatool. The documentation is split into the following sections:

  • Start: this page and glossary
  • About: learn about the idea behind Tatool
  • Community: connect with other people using Tatool and find help
  • Modules: introduction to the basic functionality
  • Task Library: reference of all available tasks and how to use them
  • Development: learn how to create your own tasks
  • Reference: find answers to specific functions and attributes while developing
