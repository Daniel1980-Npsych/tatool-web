Language Use and Cognition (UZH-LuCo)

Monitoring

This set of tasks measures monitoring of elements in working memory with the monitoring paradigm modelled after Oberauer, Süß Wilhelm, & Wittman (2003) (see also von Bastian & Oberauer, 2013; von Bastian, Souza, & Gade, 2016). The task is to monitor independently changing objects and react whenever a predefined relateion between the objects occurred. In the Squares Monitoring Task, dots are changing their position in a 10x10 grid every 2s. The task is to press the spacebar whenever four dots formed a square. In the Digits Monitoring Task, three-digit numbers are presented in a 3x3 grid, one of which changes every 2s. The task is to press the spacebar whenever the last digit of three numbers in a row, column or on a diagonal was identical.

Squares Monitoring Task

Executable

Use the following settings in the Module Editor for these tasks:

Project uzh-luco-battery
Executable Type lucoMonitoringFigural

Properties

The following Properties can be provided in the Module Editor to change the behaviour of the task (note that the default value is the one used in the publication):

Property Name Property Type Default Mandatory Description
stimuliFile Resource None, see Stimuli Files below for options Yes A stimuli file expecting stimuli according to the Tatool Stimulus definition.
stimuliPath Path None, use the following:
Project: uzh-ef-battery
Type: stimuli		 Yes A stimuli path where to load any resources from (stimuli images, input key images).
displayEncodingDuration String 6500 No Timer duration in ms to display memoranda in the encoding phase.
intervalEncodingDuration String 250 No Timer duration in ms to wait and present a blank screen before the updating phase starts.
displayUpdatingDuration String 1500 No Timer duration in ms to wait for user feedback before stopping.
intervalUpdatingDuration String 250 No Timer duration in ms to wait and present a blank screen before the next updating step.

The following stimuli files were used in the publication:

Task Project Type File Name Description

Test Trials uzh-luco-battery stimuli luco_monitoring_fig.csv 128 test trials.
