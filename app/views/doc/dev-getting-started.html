Getting Started

In order to develop your own tasks or run your own instance of tatool in your lab, you'll have setup your environment first. The following easy steps should guide you through the process of setting up your local development/lab instance of tatool.

  • Install nodeJS
  • Install mongoDB
  • Install tatool
  • Run tatool

      • Install nodeJS



      Get the latest download for your operating system from http://nodejs.org/ and follow the instructions to install it.

      Type the following command in your terminal window and make sure you get a response indicating your version of nodeJS. On windows you can enter the command by opening the Windows command prompt (cmd.exe) and on Mac you can fire up a terminal. If you don't see the version of your nodeJS, something went wrong with your installation.

      $> node --version
v0.10.33

      Install mongoDB



      Get the latest download for your operating system from http://www.mongodb.org/downloads and follow the instructions to install it. Follow the steps under 'Install MongoDB' and Step 1 under 'Run MongoDB' which will create the necessary data directory for mongoDB.

      Type the following command in your terminal window and make sure you get a response indicating your version of mongoDB. On windows you can enter the command by opening the Windows command prompt (cmd.exe) and on Mac you can run the commands in a new terminal. If you don't see the version of your mongoDB, something went wrong with your installation.

      $> cd mongodb
$mongodb> mongo -version
MongoDB shell version: 2.6.5

      Install tatool



      Get the latest download of tatool from our github repository. If you're familiar with Git, we recommend you clone the tatool-web repository, otherwise you can simply download a ZIP file by clicking on the "Download ZIP" button on the right hand side menu on github.

      After having downloaded/extracted the tatool-web repository to where you want to run tatool from, you have to run a few commands to properly install all dependencies.

      1. Open the folder where you downloaded and extracted tatool-web to in your terminal window. On windows you can enter the commands by opening the Windows command prompt (cmd.exe) and on Mac you can run the commands in a new terminal.

      $> cd tatool-web

      2. Install the bower plugin for nodejs, which will take care of downloading the application dependencies.
      Type the following command in your terminal window. 

      $tatool-web> npm install -g bower
...
$tatool-web> bower -version
1.3.12

      3. Run npm install to automatically download all dependencies needed for tatool-web to run.

      $tatool-web> npm install
...

      Run tatool

      The following steps are needed to run your local tatool-web environment. You'll need to follow these steps everytime you want to start your local instance of tatool-web.

      1. Start mongoDB by entering the command below if you haven't already started it in an earlier step. Don't close the terminal, as this will kill the process running your database.

      $> cd mongodb
$mongodb> mongod
...waiting for connections on port 27017

      2. Start nodejs in a new terminal by entering the commands below. First you'll have to change into the directory where you extracted tatool-web to. The second command will start a local web server hosting tatool. Don't close the terminal, as this will kill the process running your server.

      $> cd tatool-web
$tatool-web> npm start
...
Running tatool in LAB mode.
You can access tatool on hostname:3000

      3. Access tatool by opening the url http://127.0.0.1:3000 in your browser. You should now see the tatool start page.
