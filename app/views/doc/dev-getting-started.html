Getting Started

In order to develop your own tasks or run your own instance of tatool in your lab, you'll have setup your environment first. The following easy steps should guide you through the process of setting up your local development/lab instance of tatool.

  • Install nodeJS
  • Install mongoDB
  • Install tatool
  • Run tatool

      • Install nodeJS



      Get the latest download for your operating system from http://nodejs.org/ and follow the instructions to install it.

      Type the following command in your terminal window and make sure you get a response indicating your version of nodeJS. On windows you can enter the command by opening the Windows command prompt (cmd.exe) and on Mac you can fire up a terminal. If you don't see the version of your nodeJS, something went wrong with your installation.

      $> node --version
v0.10.33

      Install mongoDB



      Get the latest download for your operating system from http://www.mongodb.org/downloads and follow the instructions to install it. Make sure not to forget to setup the data directory as described in the installation instructions.

      Type the following command in your terminal window and make sure you get a response indicating your version of nodeJS. On windows you can enter the command by opening the Windows command prompt (cmd.exe) and on Mac you can fire up a terminal. If you don't see the version of your mongoDB, something went wrong with your installation.

      $> mongo -version
MongoDB shell version: 2.6.5

      Install tatool



      Get the latest download of tatool from our github repository. If you're familiar with Git, we recommend you clone the tatool-web repository, otherwise you can simply download a ZIP by clicking on the button on the right.

      After having downloaded/extracted the tatool-web repository to where you want to run tatool from, you have to run a few commands to properly install all dependencies.

      1. Open the folder where you downloaded and extracted tatool-web to in your terminal window. On windows you can enter the command by opening the Windows command prompt (cmd.exe) and on Mac you can fire up a terminal.

      $> cd tatool-web

      2. Install the bower plugin for node, which will take care of downloading the application dependencies.
      Type the following command in your terminal window. 

      $> npm install -g bower
...
$> bower -version
1.3.12

      3. Run npm install to automatically download all dependencies needed for tatool-web to run.

      $> npm install
...

      Run tatool

      The following steps are needed to run your local tatool-web environment. You'll need to start mongoDB and nodeJS to run tatool-web.

      1. Start mongoDB by entering the command below, which will start the database server. The terminal should not be closed, as the database will stop running.

      $> mongod
...waiting for connections on port 27017

      2. Start nodeJS/tatool in a new terminal by entering the commands below. First you'll have to change into the directory where you extracted tatool-web to. The second command will start a local web server hosting tatool. The terminal should not be closed, as the server will stop running.

      $> cd tatool-web
$tatool-web> node server.js
...Express server listening on port 3000

      3. Access tatool by opening your browser and entering the url http://127.0.0.1:3000. You should now see the tatool start page.
