Building Tatool Web

Tatool Web uses gulp for nodejs for the build process automation. Details about the tasks (concat, minify, copy) in the build process can be found in gulpfile.js.

$> gulp build

Run Tatool in lab mode, which deactivates the registration and email features and adds a default admin user.

$> npm start

# alternatively you can call node directly
$> node server.js lab prod

If you're looking to run Tatool Web on a webserver or in development mode, you'll have to provide the following two configuration parameters:

# [mode] : lab / server
# [environment] : dev / prod
$> node server.js [mode] [environment]
