Development

So you want to learn how to develop your own tasks and modules? You've come to the right place. You'll have to get familiar with the Tatool libraries and setup your local tatool environment. This section of the documentation will guide you through these steps.

You don't have to develop or setup anything, if you're fine with using the built-in tatool Module Editor with the pre-existing tasks. Take a look at the Create Modules section to learn more.
