Instruction Task

The Instruction task can be used to display image/HTML files displaying arbitrary content. The main use case would be the display of a series of images/pages instructing the user on the goal of the next task. The Instruction task offers a pagination navigation allowing a user to either go to the next or previous image/page or jump to a specific image/page. The Instruction task currently doesn't allow HTML pages from external resources.

Executable

Use the following settings in the Module Editor to use this type of task.

Project tatool
Executable Type tatoolInstruction

Properties

The following Properties can be provided in the Module Editor to change the behaviour of the task. An Instruction task can either have a pages or an images array but not both.

Property Name Property Type Default Mandatory Description
images Array (Resource) - (Yes) An array of project/external image resources (png, jpg, gif). The sequence of the items in the Array represents the display sequence.
pages Array (Resource) - (Yes) An array of project HTML resources. External HTML Resources are currently not supported. The sequence of the items in the Array represents the display sequence.

Stimuli File

This task does not support the use of a Stimuli File.

HTML customization

The Intsruction task offers a set of template variables that can be embedded in your HTML instructions to allow for dynamic content. The following template variables are currently supported:

Variable Name Description
moduleName The name of the current Module.
moduleAuthor The author of the current Module.
moduleVersion The version of the current Module.
sessionNr The current session id.
moduleProperties JavaScript object representing all available moduleProperties. In order to access the Module Property currentLevel of a LevelHandler with the name level you would use moduleProperties.level.currentLevel.

Example HTML

<div id="tatoolFormat">

  <h1>Welcome to {{moduleName}}</h1>
  
  <p>
    <center>
      You're currently in Session <span class='highlight_red'>{{sessionNr}}</span> of 10 Sessions.
    </center>
  </p>

  <img ng-src="welcome.gif">

</div>
