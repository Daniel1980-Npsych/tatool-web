Complex Span Task

The Complex Span allows you to apply the Memory span paradigm.

  • Properties
  • Stimuli File
  • Download

Properties

The following Properties can be provided in the Module Editor to change the behaviour of the task.

Property Name Property Type Default Mandatory Description
stimuliFile Resource - Yes A stimuli file expecting stimuli according to the Tatool Stimulus definition
stimuliPath Path - Yes A stimuli path where to load any resources from (stimuli images, input key images)
timerEnabled Boolean true No Enables or disables timer used to display memoranda. Should always be enabled as disabling it will not allow the task to continue.
displayDuration String 800 No Display duration for memoranda.
intervalDuration String 400 No Interval duration between memoranda.
randomisation String (none) No Defines the randomisation criteria. If no value is set no randomisation is applied. The following values are allowed:
full: fully randomises stimuli (no duplicates until all stimuli are shown once)

Stimuli File

This task requires a Stimuli File to run. In order to allow n Memoranda per Trial you can use the usual Stimulus properties and append a counter at the end (e.g. stimulusValue1). The Stimuli File should contain the following columns.

Column Name Column Values Mandatory Column Description
stimulusCount (String) Yes Number of memoranda in this trial.
stimulusValue[n] (String)/(Image Filename) Yes Value to display on screen. Can either be a text or an image.
stimulusValueType[n] text/image/circle/square Yes The stimulus value type in order to identify how to display the value.
correctResponse[n] (String) Yes The correct response for this memoranda.
