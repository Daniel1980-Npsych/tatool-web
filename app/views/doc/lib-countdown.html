Countdown Task

The Countdown tasks display a numerical countdown you can use as a starting task for your Executables to make sure the user is ready.

  • Executable
  • Properties
  • Stimuli File
  • Download

Executable

Use the following settings in the Module Editor to use this type of task.

Project tatool
Executable Type tatoolCountdown

Properties

The following Properties can be provided in the Module Editor to change the behaviour of the task.

Property Name Property Type Default Mandatory Description
countdown String 5 No The number to start to count down from.
interval String 600 No Interval used to refresh countdown numbers.
goText String 'Go!' No Text to display once countdown has finished.

Stimuli File

This task does not support the use of a Stimuli File.
