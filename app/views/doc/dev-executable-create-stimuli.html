Create Stimuli

In this section we're going to create a stimuli file and the corresponding stimuli to be used by our Executable.

A stimuli file represents the list of stimuli and their properties you'd like to use in your Executable. A stimulus can be anything from a single character, a word, a digit, an image, a video or an audio file. As long as you can show/play the stimulus on a HTML page, it should be fine to use with Tatool. The stimuli file should be a CSV (Comma-separated values) file.

In order for the CSV file to be used with Tatool, you should follow these conventions:

  • Use either comma (,) or semicolon (;) as your delimiter. Tatool will try to detect the delimiter automatically.
  • Use a header line, as it will simplify the access to the different columns of your stimulus later on.
  • Use lowerCamelCase for the header columns, as it will simplify the access to the different columns of your stimulus later on.

For our purpose let's create a stimuli file for a Flanker Task. Therefore, a stimulus is defined as being congruent, incongruent or neutral (stimulusType). We'll need to define whether the items in our stimulus are going to be text or image values (stimulusValueType) (e.g. character, digit, arrow) and what their values (stimulusValue) are going to be (e.g. 'A', 'congruent-left.png'). As a last property, we'll have to define the correct response (correctResponse) in order to compare the response given by the user with the expected response to provide some user feedback.


Example arrow stimuli

We end up with a stimuli file that will look similar to this:

stimulusType;stimulusValue;stimulusValueType;correctResponse
congruent;congruent-right.png;image;right
incongruent;incongruent-right.png;image;right
neutral;neutral-left.png;image;left
incongruent;incongruent-right.png;image;right
congruent;congruent-right.png;image;right
incongruent;incongruent-left.png;image;left
neutral;neutral-right.png;image;right
neutral;neutral-left.png;image;left
incongruent;incongruent-left.png;image;left
neutral;neutral-left.png;image;left
...

You can create your own stimuli file or use the example already provided in your local Tatool installation. At the same time we also have to create our actual stimuli which we've decided are going to be images. There's going to be a total of 6 images for every possible combination of left/right and congruent/incongruent/neutral stimuli types. You again can create your own images or use the ones provided in your local Tatool installation.


Example stimulus image (incongruent-left.png)

In order to use the stimuli files in our Executable, we'll have to save/copy all stimuli related files into our project stimuli sub-folder (app/projects/public/myExperiment/stimuli) of our local Tatool installation.
