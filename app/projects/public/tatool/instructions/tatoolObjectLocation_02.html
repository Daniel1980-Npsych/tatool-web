Instruction

As soon as all animals of one run have been shown, the grid will be cleared and all animals that appeared will be available in a second grid below the main grid.
It's your task now to drag and drop the animals from the bottom grid to the position where they appeared in the main grid. Think before you move, as you can't change the position of an animal once it's been dropped in the main grid. Once you've dropped the last animal, you'll receive a visual feedback indicating which animals were placed correctly (green) and which animals were placed wrongly (red).



Start the task by pressing the right arrow key on your keyboard.
You can exit the task at anytime by pressing the escape key.
